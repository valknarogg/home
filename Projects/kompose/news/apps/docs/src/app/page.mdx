# Introduction

LetterSpace is an open-source newsletter management platform focused on user experience. It's designed to be powerful yet simple to set up and use.

Creating newsletters shouldn't be difficult or expensive. LetterSpace provides a self-hosted solution that helps you build your audience quickly without unreasonable costs.

<Callout type="warning">
  This project is in early development. Some features may be incomplete or
  subject to change.
</Callout>

## Key Features

- Multiple newsletter management
- Subscriber management
- Campaign creation with analytics
- Template management
- REST API for easy integrations
- Configurable SMTP settings
- Easy setup and deployment with Docker
- WYSIWYG email editor (WIP)

We welcome contributions to help improve LetterSpace.

Head to [Getting Started](/getting-started) to launch your newsletter in minutes.
